О нас

Мы верим, что в основе лучших приложений лежат простые идеи, уникальный дизайн и интуитивная модель взаимодействия с пользователем. Удовольствие от использования наших продуктов и хорошее впечатление от работы с нами ­ главное, что ценят наши клиенты.

Кто мы

Мы молодая и динамичная компания, расположенная в городе Стерлигн, штат Вирджиния. Наша компания занимается разработкой мобильного программного обеспечения, используя самые современные технологии, доступные на рынке. В числе наших клиентов находятся как крупные финансовые организации, так и перспективные стартап проекты.

Наши проекты

Мы работаем как в области мобильных финансовых решений, так и в области перспективных проектов. Такое соотношение позволяет нам добиться баланса между потоком денежных средств и использованием новейших и экспериментальных технологий. Это дает нам возможность постоянно развиваться, изучать новое и раскрывать наши творческие таланты.

Используемые Технологии

Основной платформой является Apple, используем язык программирования Swift. Для гибридных приложений используем Ioinic, Xamarin. Для тестирования наших продуктов мы используем Frank, Symbiote, Sauce Lab. Для гибридных приложений Karma, SitePrism, Protractor. Облачные решения построены с использованием NodeJS/KoaJS/MongoDB/FoundationDB/Redis. Инфраструктурные решения на основе Terraform, Puppet, Consul, Docker.

Мы используем в работе
Docker

Docker is an open platform for developers and sysadmins to build, ship, and run distributed applications. Docker enables apps to be quickly assembled from components and eliminates the friction between development, QA, and production environments.

Consul

Consul has multiple components, but as a whole, it is a tool for discovering and configuring services in your infrastructure. It provides Service Discovery, Health Checking and Key/Value store.

Terraform

Terraform is a tool for building, changing, and versioning infrastructure safely and efficiently. Terraform can is a code-over-infrastructure with multi data center support.

Swift language

Swift is an innovative new programming language for Cocoa and Cocoa Touch. Swift is ready for iOS and OS X projects. Swift code works side-by-side with Objective-C.

Ionic

The world's first full-stack hybrid mobile platform. Allows to build amazing apps on any platform with one single code base.

NodeJS

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.
